# VerifAI
The solution to giving an analysis on AI use in video production based on news articles revolving key search terms.

# Setup
## Requirements
- Node.js (v18+)
- npm
- Git
- Google Gemini API Key
- TheNewsAPI Key

## Cloning
```bash
git clone https://github.com/MichaelTahirovic/SpurHacks2025.git
```

## Backend Setup
```bash
cd backend
npm install
node server.js
```

## Frontend Setup
```bash
cd frontend
npm install
npm start
```

# Technologies Used
- React.js
- Node.js
- Express.js
- Google Gemini API
- TheNewsAPI
